# SARTHI_Rover
(Major Project)
SARTHI agriculture rover helps farmers to targeted fertilization process of solid fertilizers. Features autonomous navigation in farm land and deliver precie amount of  fertilizer near to the stem of crop using 4-DOF robotic arm implemented on it.

Sensors Used:
  1. USB Webcameras(Stereo Camera Setup)
  2. Load cell 5KG
  3. Ultrasonic sensors
  4. Neo-6M GPS sensors
  5. Magnetometer sensor
  6. IMU sensor ( MPU6050 / ADXL345 )

Control Boards:
  1. Rpi- 8GB 128GB
  2. MCU
